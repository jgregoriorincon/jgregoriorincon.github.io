Índice de Seguridad y Convivencia Barrial

  •  Acerca del Proyecto
  •  Advertencia

Un mapa de Información del indice de seguridad de la ciudad de Santa Marta.

Esta aplicación esta construida utilizando el framework Bootstrap 3, Leaflet, y typeahead.js.

Caracteristicas
  • Mapa a pantalla completo ajustado a dispositivos moviles con una barra de navegación responsive
  • Carga de datos GeoJSON usando jQuery
  • Manejo de datos en cluster de manera dinamica utilizando el plugin leaflet marker cluster
  • Manejo de busquedas inteligentes y autocompletado usando typeahead.js
  • Listado de atributos responsive, sincronizando limites del mapa a través de list.js
  • Iconos de elementos disponibles usando el pligin grouped layer control

Los datos de este sitio son propiedad exclusiva de la ODSC de Santa Marta y estan disponibles solo para propositos informativos.

Ninguna precisión, exactitud o completitud en los datos se encuentra gararantizada o implica obligatoriedad de la ODSC Santa Marta. Toda información en este mapa se encuentra sujeta a cambios, variaciones y correcciones de acuerdo a las situaciones que se presenten con los mismos.
